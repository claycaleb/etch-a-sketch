# Etch-A-Sketch

A browser-based drawing application inspired by the classic Etch-A-Sketch toy, built with vanilla HTML, CSS, and JavaScript as part of [The Odin Project](https://www.theodinproject.com/) curriculum.

## Features

- Interactive drawing grid
- Dynamic grid generation with JavaScript
- Mouse hover effects for drawing
- DOM manipulation practice

## Technologies

- HTML
- CSS
- JavaScript

## Status

⚠️ **Learning Project** - This was an early exercise in DOM manipulation and event handling. Archived for reference.
